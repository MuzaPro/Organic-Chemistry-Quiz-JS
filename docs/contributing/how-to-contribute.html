Contributing to the Organic Chemistry Quiz

Thank you for your interest in contributing to the Organic Chemistry Quiz! As a community-driven educational tool, we particularly value contributions from chemists, educators, and researchers who can help expand our question bank with accurate, pedagogically sound content.

While our Developer Guide covers code contributions, this page focuses on how subject matter experts can contribute chemistry questions without necessarily needing programming expertise.

Why We Need Your Chemistry Expertise

Creating effective chemistry questions requires both:

  • Deep understanding of organic chemistry concepts
  • Pedagogical insight into common student misconceptions
  • Ability to craft explanatory content that clarifies difficult concepts

Your expertise as a chemist or educator is invaluable in creating questions that are both technically accurate and educationally effective.

Question Bank Format

Our questions are stored in a structured JSON format. Each question includes:

  • Basic information: Question ID, text, and reaction type
  • Product details: The target molecule students need to synthesize
  • Reagent options: Both correct and incorrect reagents for students to choose from
  • Feedback: Educational responses for both correct and incorrect answers
  • Chemistry notes: Detailed explanations of the reaction mechanism and theory

How to Contribute Questions

Option 1: Direct JSON Contribution (Technical)

If you're comfortable with JSON formatting:

  1. Fork the repository on GitHub
  2. Add your questions to the data/questions.json file following our format
  3. Submit a pull request with your additions

Option 2: Simplified Submission (Non-Technical)

If you prefer not to work directly with JSON:

  1. Create a document (Word, Google Doc, etc.) with your question details
  2. Submit it via email to studio@muzascience.com or open an issue on GitHub
  3. Our team will convert your submission into the proper format

Using AI Tools to Format Contributions

You can use AI tools like ChatGPT to help format your questions into our JSON structure:

  1. Write your question, correct answer, wrong answers, and explanations in plain text
  2. Ask the AI to format it according to our JSON schema (share the example above)
  3. Review the output for accuracy before submitting

Question Requirements

Essential Components for Each Question

  • IUPAC Names: All molecules (reagents and products) must include proper IUPAC names
  • Molecular Formulas: Include accurate chemical formulas
  • Text Representations: Simple text-based molecular representations
  • Detailed Feedback: Educational responses for both correct and incorrect answers
  • Chemistry Notes: In-depth explanation of the reaction mechanism, theory, and practical applications

Types of Reactions We're Looking For

While we welcome all types of organic chemistry reactions, we're particularly interested in expanding our coverage of:

  • Pericyclic reactions (Diels-Alder, cycloadditions)
  • Transition metal-catalyzed reactions
  • Carbohydrate and amino acid chemistry
  • Stereochemistry-focused problems

If you're using Option 2 (non-JSON submission), please include the following information:

  • Reaction Type: [E.g., "SN2", "Aldol Condensation", etc.]
  • Product:

    • Formula: [Chemical formula]

Review Process

All question submissions go through a review process:

  1. Initial screening for format completeness
  2. Technical review for chemical accuracy
  3. Educational review for pedagogical effectiveness
  4. Final formatting and inclusion in the question bank

We may contact you with questions or suggestions to improve your submission.

Example Submission

Here's an example of a well-formatted question submission:

Example: Aldehyde Reduction Question

Question Text: Select reagents for this aldehyde reduction reaction:

Reaction Type: Reduction

Product:

  • IUPAC Name: 1-Phenylethanol
  • Formula: C8H10O
  • Text Representation: C6H5-CHOH-CH3

Correct Reagents:

  • IUPAC Name: Acetophenone (1-phenylethanone)
  • Formula: C8H8O
  • Text Representation: C6H5-CO-CH3
  • IUPAC Name: Sodium borohydride
  • Formula: NaBH4
  • Text Representation: NaBH4

Incorrect Reagent Options:

  • IUPAC Name: Lithium aluminum hydride
  • IUPAC Name: Benzaldehyde
  • IUPAC Name: Hydrogen (with palladium catalyst)
  • IUPAC Name: Sodium hydroxide

Correct Answer Feedback: Correct! Sodium borohydride (NaBH4) is a mild reducing agent that effectively reduces ketones like acetophenone to secondary alcohols. In this case, acetophenone is reduced to 1-phenylethanol.

Incorrect Answer Feedback: Incorrect. Ketone reduction to a secondary alcohol requires a suitable reducing agent. Consider which reagent can deliver hydride to the carbonyl carbon while being mild enough not to reduce other functional groups.

Chemistry Notes: <p>The <strong>Reduction of Ketones</strong> to secondary alcohols is a fundamental transformation in organic synthesis:</p><ul><li>Sodium borohydride (NaBH4) is a mild, selective reducing agent</li><li>It delivers a hydride nucleophile to the electrophilic carbonyl carbon</li><li>The reaction is typically performed in protic solvents like methanol or ethanol</li><li>NaBH4 reduces aldehydes and ketones but is not strong enough to reduce esters, amides, or carboxylic acids</li></ul><p>The mechanism involves:</p><ol><li>Nucleophilic attack of hydride on the carbonyl carbon</li><li>Formation of an alkoxide intermediate</li><li>Protonation by the solvent to form the alcohol product</li></ol><p>This reaction produces a stereogenic center when the ketone is asymmetric, typically giving a racemic mixture unless chiral reducing agents are employed.</p>
